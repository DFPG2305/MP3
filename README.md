# MP3
## Commit 1
* Se agregó la selección manual de sacrificios para monstruos de nivel alto. Ahora el jugador puede elegir qué monstruo del campo desea sacrificar al invocar cartas que requieren tributo, en lugar de sacrificar automáticamente el primer monstruo disponible.

* Se añadieron descripciones para cartas mágicas y cartas trampa mediante un método getDescripcion() implementado con polimorfismo.

* Se modificó la ventana de “Jugar Carta” para mostrar información descriptiva de cartas mágicas y trampas antes de ser jugadas, con el fin de orientar al jugador sobre sus efectos.

## Commit 2
*Se introdujeron 3 clases con el objetivo meramente estetico de que se pudieran apreciar las cartas en el campo de batalla, esto con el objetivo de una mayor conexion con el titulo pues se usan todas las imagenes de las cartas originales

*Se agrego el archivo PDF "Minitutorial" el cual contiene las fuentes que se usaron para desarrollar la parte estetica de las cartas, adicionalmente de explicar el como funcionaban cada una de estas clases y el porque se tomo la desicion de utilizarlas
